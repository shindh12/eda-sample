# EDA Sample Project

-----------------
## Requirement
### 도서 관리 시스템
- 전체 도서를 조회 할 수 있다.
- 대여 가능한 도서를 조회 할 수 있다.
- 도서 대여를 할 수 있다.
- 도서 대여를 취소할 수 있다.
- 도서 반납을 할 수 있다.
- 내가 대여한 도서 목록을 조회할 수 있다.
- 도서 대기알림을 받을 수 있다. (도서 대여가 불가능 할 경우)
- 독서 순위 TOP5를 조회할 수 있다.

## Non-functional Requirement
- 도서 대여는 수량 이상으로 동시에 대여 될 수 없다.
- 독서 순위의 정합성이 잘 맞아야 한다.
